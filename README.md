# Judo Federation of Deaf Athletes of Kazakhstan

Multilingual responsive website created for Software Project 2.

## Languages

- Russian: `index.html`
- Kazakh: `index-kz.html`
- English: `index-en.html`

## How to run

Open `index.html` in a web browser.

For full functionality, the project can also be started with a local server.

## Technologies

- HTML
- CSS
- JavaScript

## Main features

- Responsive design
- Russian, Kazakh and English versions
- Championship galleries
- Image sliders
- Mobile navigation
- Accessible keyboard controls
